⚠️ Aviso

Este script move arquivos automaticamente com base em suas extensões.
Recomenda-se utilizá-lo em pastas de teste ou fazer backup antes da execução, para evitar perda ou reorganização indesejada de arquivos.

Funcionalidades

Organiza arquivos automaticamente por tipo
Cria pastas automaticamente, caso não existam
Suporte para:
📷 Imagens (.jpg, .png, .jpeg)
📄 Documentos (.pdf, .txt, .docx)
🎥 Vídeos (.mp4, .mkv)
🎵 Áudios (.mp3, .wav)
Arquivos não reconhecidos são movidos para a pasta "Outros"
🛠️ Tecnologias Utilizadas
Python 3
Biblioteca os
Biblioteca shutil
▶️ Como Executar

1. Acesse a pasta do projeto
2. Execute o script
python organizador.py
3. Informe o caminho da pasta

Exemplo:

C:\Users\SeuNome\Downloads
Estrutura após execução

Exemplo de organização:

Downloads/
│
├── Imagens/
├── Documentos/
├── Videos/
├── Audios/
└── Outros/
